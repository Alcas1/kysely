Class InsertResult

The result of an insert query.

If the table has an auto incrementing primary key insertId will hold the generated id on dialects that support it. For example PostgreSQL doesn't return the id by default and insertId is undefined. On PostgreSQL you need to use ReturningInterface.returning or ReturningInterface.returningAll to get out the inserted id.

Examples

 
const result = await db
  .insertInto('person')
  .values(person)
  .executeTakeFirst()

console.log(result.insertId)
