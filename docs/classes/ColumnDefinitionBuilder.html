defaultTo

  • defaultTo(value: unknown): ColumnDefinitionBuilder

  • Implementation of ColumnDefinitionBuilderInterface.defaultTo

    • Defined in src/schema/column-definition-builder.ts:278

    Adds a default value constraint for the column.

    Examples

     
    db.schema
      .createTable('pet')
      .addColumn('number_of_legs', 'integer', (col) => col.defaultTo(4))
      .execute()

    Values passed to defaultTo are interpreted as value literals by default. You can define an arbitrary SQL expression using the sql template tag:

    import { sql } from 'kysely'

    db.schema
      .createTable('pet')
      .addColumn(
        'number_of_legs',
        'integer',
        (col) => col.defaultTo(sql`any SQL here`)
      )
      .execute()
