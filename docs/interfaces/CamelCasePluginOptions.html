Optional underscoreBeforeDigits

underscoreBeforeDigits?: boolean
  • Defined in src/plugin/camel-case/camel-case-plugin.ts:33

If true, an underscore is added before each digit when converting camelCase to snake_case. For example foo1Bar => foo_1_bar and foo_1_bar => foo1Bar

Defaults to false.

Optional underscoreBetweenUppercaseLetters

underscoreBetweenUppercaseLetters?: boolean
  • Defined in src/plugin/camel-case/camel-case-plugin.ts:42

If true, an underscore is added between consecutive upper case letters when converting from camelCase to snake_case. For example fooBAR => foo_b_a_r and foo_b_a_r => fooBAR.

Defaults to false.

Optional upperCase

upperCase?: boolean
  • Defined in src/plugin/camel-case/camel-case-plugin.ts:24

If true, camelCase is transformed into upper case SNAKE_CASE. For example fooBar => FOO_BAR and FOO_BAR => fooBar

Defaults to false.
