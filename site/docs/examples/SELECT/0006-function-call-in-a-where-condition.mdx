# Function call in a WHERE condition

If you need to call a function in a WHERE statement (or anywhere, really) you can use the 
[fn](https://kysely-org.github.io/kysely/interfaces/FunctionModule.html) property of the
[ExpressionBuilder](https://kysely-org.github.io/kysely/interfaces/ExpressionBuilder.html).
